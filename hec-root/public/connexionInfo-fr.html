Comment ouvrir une session :

Communauté HEC

  1. Entrez votre matricule HEC (employé, étudiant ou diplômé) dans la zone « Code d’utilisateur ».
  2. Entrez votre mot de passe centralisé HEC. Votre mot de passe est le même que pour les applications de courrier électronique et d’agenda de HEC Montréal.

Compte Invité HEC

  1. Entrez votre adresse courriel.
  2. Entrez le mot de passe qui vous a été envoyé par courriel.

Problèmes :

Communauté HEC

Si vous êtes étudiant ou diplômé vous pouvez réinitialiser votre mot de passe, sinon, pour tout autre problème, veuillez contacter :

Le Centre d'assistance du lundi au vendredi entre 8:00 a.m. et 10:00 p.m..

  • En personne: bureau 3.820 (édifice Côte-Sainte-Catherine)
  • Par téléphone: 514-340-6063
  • Par courriel: soutien.ti@hec.ca

Compte Invité HEC

Si vous éprouvez des problèmes, communiquez avec l'enseignant qui vous a invité en lui demandant de supprimer votre nom du site de cours et de vous réinviter.
